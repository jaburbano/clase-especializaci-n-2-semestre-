# Clase Especialización 2° Semestre

Este repositorio reúne material de estudio, ejercicios prácticos y configuraciones utilizadas en la especialización en Big Data.

Objetivo
- Servir como base práctica para el aprendizaje y la experimentación con herramientas y técnicas de Big Data.

Qué contiene
- Apuntes y presentaciones por módulo.
- Notebooks y scripts de ejemplo (Spark, PySpark, Hadoop, Kafka, etc.).
- Configuraciones de infraestructura y plantillas para despliegue.
- Ejercicios y datasets de práctica.

Cómo usarlo
- Clona este repositorio y explora las carpetas por módulo.
- Abre los notebooks con Jupyter/Colab para reproducir los ejercicios.
- Revisa las carpetas de infraestructura para ejemplos de configuración.

Contribuciones
- Puedes abrir issues para reportar problemas o sugerir mejoras.
- Si quieres aportar contenido, crea un fork y envía un pull request.

Contacto
- Autor/Responsable: Juan A. Burbano (o la persona encargada del curso).

Este repositorio está pensado para apoyar el aprendizaje durante el segundo semestre de la especialización en Big Data y puede actualizarse regularmente con nuevo material.
